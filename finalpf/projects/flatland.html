This activity demonstrates how a static layout becomes a dynamic interactive experience in 8 progressive steps.

  1. Step 1: Understand the Basic Structure
  2. Step 2: Add Simple Interactivity
  3. Step 3: Add Hover & Mouseout Events
  4. Step 4: Use a Parameterized Function
  5. Step 5: Use addEventListener()

Reflection

This activity helped me understand how JavaScript can progressively enhance a webpage. By going from inline events to external scripts, I learned how to properly separate concerns between HTML, CSS, and JavaScript. The use of DOM manipulation, events, and dynamic content updates has improved my confidence in building interactive web pages. The final version of Flatland reflects a clean, maintainable, and scalable approach that aligns with industry best practices.
